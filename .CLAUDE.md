# Project: Dory (Android)

## Environment

- Java/JDK is bundled with Android Studio:
  `JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home"`
- Set `JAVA_HOME` before running Gradle commands (e.g. `./gradlew test`)

## Workflow

- Always use git worktrees for feature work. Never commit directly to `main`.
- Create worktrees under `.worktrees/` (e.g., `.worktrees/feature-name`).

## Allowed commands

- Read-only commands: git status, git log, git diff, git branch, git worktree list, ./gradlew test, ls, cat, head, tail, find, grep, rg, wc, file, which
